🚀 BunkZone

BunkZone is a simple web application that helps students calculate how many classes they can bunk 
while maintaining the required attendance percentage.

📌 Features
🎯 Calculate safe bunk days instantly
📊 Shows current attendance status
⚡ Fast and easy to use
💻 Works on all devices (responsive design)
🧮 Real-time calculation

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript

📥 How It Works
Enter total classes held
Enter classes attended
Click on Calculate
Get:
Attendance percentage
Number of classes you can bunk

🌐 Live Demo

https://bunkzone.vercel.app/

📊 Formula Used
Attendance % = (Classes Attended / Total Classes) × 100

🎯 Use Case
Students maintaining 75% attendance rule
Quick calculation before bunking classes 😅
